Google Sheets integration

Connect your Google Sheet using a published Apps Script Web App URL. The dashboard will fetch live data from your sheet on every load.

Web App URL
Setup steps:
1. Open your Google Sheet
2. Extensions → Apps Script
3. Paste the doGet/doPost script
4. Deploy as Web App (access: Anyone)
5. Copy the URL and paste above
